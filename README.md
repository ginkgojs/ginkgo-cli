
## ginkgo-cli

---

A simple CLI for ginkgo.js project

### Usage

ginkgo \<command> [options]

---

### Commands

#### init
> create a project

```shell
ginkgo init my-project
```


#### controller
> add a controller for current project

```shell
ginkgo controller v2 hello
```
